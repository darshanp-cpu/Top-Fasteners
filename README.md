## ðŸš€ Deployment

### GitHub Pages
Detailed automatic deployment:

```bash
npm run deploy
```
This runs the build script and pushes the `dist/` folder to the `gh-pages` branch.

### Vercel
1. Connect this repository to Vercel.
2. Vercel will detect `package.json` and run `npm run build`.
3. **Settings**: Ensure "Output Directory" is set to `dist` (if not auto-detected).
